== Description ==

[**Limit Attempts Booster**](https://tech-banker.com/limit-attempts-booster/) was designed to manage the access to users website and provide bullet proof security to your website.

Limit Attempts Booster aims to provide login protection and  protect your website from Brute Force attacks. Its perspective is to secure your account.

With this plugin

* you can limit users attempts to login to your website
* you can block the user for a certain period of time
* users are allowed to create and edit unblock and block list with the aim of controlling the access to the site and its content
* users can specify the settings for adding an IP Address to the block list
* be aware of everything that is going on our website by sending email notifications about blocking
* admin have a comprehensive control of user activity
* you are able to protect website from automated logins efficiently
* you can send Email notifications to the Administrator
* you can control the capabilities of Limit Attempts Booster among different roles of WordPress users

You can view the following things:

* List of Blocked IP Addresses
* List of Unblocked IP Addresses
* List of Blocked IP Ranges
* List of Unblocked IP Ranges
* Login Logs with Geo Location in the Map

It gives you access to set Auto IP Block so that after certain number of unsuccessful login attempts the IP Address of unauthorized user gets blocked.

Another advanced feature is that you can block certain countries to protect your website from unauthorized access.

It gives you option to setup different alerts for different actions performed by user while login

* whether Login is Successful
* whether Login is Unsuccessful
* when IP Address is Blocked
* when IP Address is Unblocked
* when IP Ranges are Blocked
* when IP Ranges are Unblocked

**Limit Attempts Booster uses our own Tech Banker API based on Maxmind Database to fetch location and other details based on IP Address for its feature Login Logs. For this reason, we are sending back IP Address whenever a user tries to login to your site back to our server to fetch the details.**

== Installation ==

#### Thank you for your interest in Limit Attempts Booster.


### Minimum Requirements.

*   Wordpress 3.5+
*   PHP 5.x
*   MySQL 5.x

1. Upload the `limit-attempts-booster` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.


### For Mac Users


*  Go to your Downloads folder and locate the folder with the plugin.
*  Right-click on the folder and select Compress.
*  Now you have a newly created .zip file which can be installed as described here.
*  Click "Install Now" button.
*  Click "Activate Plugin" button for activating the plugin.
